# Technology Stack

This page outlines the chosen technology stack for the **Job Portal Application**, along with the stable versions.

---

## Frontend
- **React + Vite (v19.2.0 / v7.2):** Provides fast UI rendering and hot module replacement for rapid development.  
- **Redux (v5.0.1):** Manages global state across different components.  
- **Tailwind CSS (v4.1.13):** Used for responsive and modern design.  
- **Shadcn/ui (v0.9.5):** Customizable component library used for accessible UI components (like modals, dropdowns, buttons, and inputs).  

## Backend
- **Node.js (v24.x):** Provides a runtime environment for server-side logic.  
- **Express.js (v5.1.0):** Handles API routes and middleware logic.  
- **JWT Authentication (v9.0.2):** Ensures secure login and role-based route protection.
